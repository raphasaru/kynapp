    Page metadata:

      description: 'Como o KYN protege seus dados financeiros. LGPD compliant.',
    }

    Content sections (prose-style, readable, non-legalese Portuguese):

    1. **Introdução**: "O KYN leva a privacidade dos seus dados a sério. Esta política explica como coletamos, usamos e protegemos suas informações."

    2. **Dados que coletamos**:
    - Dados de cadastro: email, nome
    - Dados financeiros: transações, contas, cartões (armazenados com criptografia)
    - Dados de uso: páginas visitadas, funcionalidades utilizadas
    - Dados do WhatsApp: mensagens enviadas ao bot (processadas e descartadas)

    3. **Como usamos seus dados**:
    - Fornecer o serviço de gestão financeira
    - Processar transações via WhatsApp
    - Melhorar a experiência do app
    - NÃO vendemos, compartilhamos ou monetizamos dados pessoais

    4. **Proteção dos dados**:
    - Criptografia AES-256-GCM para dados financeiros
    - Dados criptografados antes de sair do seu navegador
    - Row-Level Security no banco de dados
    - Acesso restrito por autenticação

    5. **Seus direitos (LGPD)**:
    - Direito de acesso: saber quais dados temos
    - Direito de correção: atualizar dados incorretos
    - Direito de exclusão: deletar sua conta e todos os dados
    - Direito de portabilidade: exportar seus dados
    - Direito de revogar consentimento
    - Para exercer: enviar email para [contato@kynapp.com.br] (placeholder)

    6. **Cookies e armazenamento local**:
    - Cookies de sessão (autenticação Supabase)
    - localStorage para preferências do app
    - Sem cookies de rastreamento de terceiros

    7. **Retenção de dados**:
    - Dados mantidos enquanto a conta estiver ativa
    - Após exclusão da conta: dados removidos em até 30 dias
    - Plano Free: histórico limitado a 3 meses

    8. **Alterações nesta política**: Notificaremos por email sobre mudanças significativas.

    9. **Contato**: Email placeholder.

    10. **Data de vigência**: Fevereiro de 2026.

    Style: Use Tailwind prose classes (`prose prose-neutral max-w-3xl mx-auto`). Add back link to home (/). Simple layout without sidebar/nav. Heading font (Space Grotesk). Adequate spacing. SECU-04 badge at top showing this is a secure, transparent company.
